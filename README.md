# מסך תצוגה יוקרתי ללובי — הוד השרון

מסך מידע ללובי בניין: שעה, תאריך עברי ולועזי, מזג אוויר, אירועי השבוע (אלומה),
חדשות, וברכות חג לפי לוח השנה העברי — עם מחזור תצוגה חלק.

## הרצה מקומית
```bash
npm install
npm run dev
```

## בנייה
```bash
npm run build
```

## ניהול גרסאות
הגרסה מנוהלת בקובץ `src/version.js` (קבוע `VERSION` + `CHANGELOG`).
הגרסה הנוכחית מוצגת בתחתית המסך; לחיצה עליה פותחת את יומן הגרסאות המלא.
בכל שינוי: הוסיפו רשומה לראש `CHANGELOG` והעלו את `VERSION` (semver).

## מקורות נתונים
- **אירועים**: אלומה (`alumahod.com`) — סינון לשבוע הנוכחי, עם cache (בייצור דרך proxy).
- **חגים ולוח עברי**: `@hebcal/core` — חישוב מקומי, ללא תלות ברשת.
- **מזג אוויר**: Open-Meteo (הוד השרון) — עם fallback לערך אחרון שנשמר.
- **חדשות**: הזנת RSS רשמית (בייצור).

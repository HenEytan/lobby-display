# מסך לובי — יסוד המעלה 9, הוד השרון (v3.1.0)

מערכת תצוגה ללובי בניין מגורים + פאנל ניהול מובנה.

## ניתוב
- `/` — מסך התצוגה (ברירת מחדל)
- `/#admin` — כניסת מנהל (PIN, ברירת מחדל: `1234` — מומלץ לשנות בהגדרות)
- `/#preview` — תצוגה מקדימה של טיוטות לפני פרסום

## ארכיטקטורה
- **localStorage** — נתונים טקסטואליים (הגדרות, באנרים, הודעות, טיקר, פלייליסט) במודל טיוטה/פרסום (`lobby_*` / `lobby_draft_*`)
- **IndexedDB** (`lobby-media`) — תמונות (נדחסות אוטומטית) וקבצי אודיו
- קריאות חיצוניות: **Open-Meteo** (מזג אוויר) + **מבזקי ynet** דרך פונקציית שרת `api/ynet.js` (מתווך RSS, ללא CORS) — שתיהן עם מטמון fallback
- לוח עברי, חגים, צומות ושבת: חישוב מקומי עם `@hebcal/core` (כולל שקופית לוח שנה לשנה הקרובה)
- **ערכות צבע** מתחלפות מפאנל הניהול (`src/lib/themes.js`) — ברירת מחדל "עץ חם" תואמת קיר דמוי-עץ
- תמונות אמנות מקוריות (SVG מקומי, `src/lib/artwork.jsx`) — ללא מקורות חיצוניים

## פיתוח
```bash
npm install
npm run dev
npm run build
```

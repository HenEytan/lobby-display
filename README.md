# מסך תצוגה ללובי — הוד השרון

מסך מידע ללובי בניין: שעה, תאריך עברי ולועזי, מזג אוויר, אירועי אלומה,
חדשות מ-Ynet וברכות חג לפי לוח השנה העברי.

## הרצה מקומית
```bash
npm install
npm run dev
```

## מקורות נתונים
- **חדשות**: Ynet RSS דרך `/api/news` (serverless — עוקף CORS)
- **אירועים**: אלומה דרך `/api/aluma-events` — פירוס אוטומטי של "החודש באלומה"
- **מזג אוויר**: Open-Meteo (הוד השרון) — הקריאה החיצונית היחידה מהדפדפן
- **לוח עברי**: `@hebcal/core` — חישוב מקומי

כל מקור נשמר ב-localStorage כגיבוי במקרה נפילה.

## ניהול גרסאות
`src/version.js` — קבוע `VERSION` + `CHANGELOG`. לחיצה על תגית הגרסה פותחת את היומן.

# מסך לובי — יסוד המעלה 9, הוד השרון (v3.2.0)

מערכת תצוגה ללובי בניין מגורים + פאנל ניהול מובנה.

## ניתוב
- `/` — מסך התצוגה (ברירת מחדל)
- `/#admin` — כניסת מנהל (PIN, ברירת מחדל: `1234` — מומלץ לשנות בהגדרות). **אין קישור גלוי במסך הראשי** — יש להקליד את הכתובת ידנית בדפדפן.
- `/#preview` — תצוגה מקדימה של טיוטות לפני פרסום

## ארכיטקטורה
- **localStorage** — נתונים טקסטואליים (הגדרות, באנרים, הודעות, טיקר, פלייליסט) במודל טיוטה/פרסום (`lobby_*` / `lobby_draft_*`)
- **מיגרציה חד-פעמית** (`src/lib/store.js`, `migrateOnce`) — כשמעדכנים ברירות מחדל של המערכת (למשל הפיכת הודעה לבאנר), מכשירים שכבר פועלים בשטח מתעדכנים אוטומטית ברענון הראשון, בלי צורך בפעולה ידנית
- **IndexedDB** (`lobby-media`) — תמונות (נדחסות אוטומטית) וקבצי אודיו
- קריאות חיצוניות: **Open-Meteo** (מזג אוויר, קואורדינטות הוד השרון) + **מבזקי ynet** דרך פונקציית שרת `api/ynet.js` (מתווך RSS, ללא CORS) + **יוטיוב** (הטמעת iframe לניגון מוזיקת רקע בלולאה, אופציונלי) — כולן עם מטמון/נפילה חלקה
- לוח עברי, חגים, צומות ושבת: חישוב מקומי עם `@hebcal/core` (כולל שקופית לוח שנה לשנה הקרובה)
- **ערכות צבע** מתחלפות מפאנל הניהול (`src/lib/themes.js`) — ברירת מחדל "עץ חם" תואמת קיר דמוי-עץ
- **תמונות אמנות מקוריות** (SVG מקומי, `src/lib/artwork.jsx`) — ללא מקורות חיצוניים:
  - 5 רקעי באנר מאוירים (שקיעה עירונית, גינה פורחת, תבנית גיאומטרית, קיץ ובריכה, ניקיון ותחזוקה) — באנר ללא רקע נבחר ידנית **מתחלף אוטומטית כל 3 ימים** בין כל האיורים (`rotatingArt`)
  - אייקוני קטגוריה מאוירים לכרטיסי אירועים (מופעים/משפחות/תרבות/קהילה) והודעות דיירים (ועד/תחזוקה/קהילה/חגיגי)
- **מוזיקת רקע** — שני מקורות לבחירה בניהול:
  - יוטיוב: קישור לוידאו, ניגון בלולאה אינסופית ברקע (ללא תצוגת וידאו)
  - קבצים שהועלו: MP3 בפלייליסט, מאוחסן ב-IndexedDB
  - כשפעילה, מצטרפת שקופיית "מנגן כעת" למחזור התצוגה הראשי

## מחזור שקופיות ראשי
באנרים פעילים ← אירועי אלומה השבוע ← לוח חגים ומועדים ← ברכת חג (אם רלוונטי) ← מנגן כעת (אם מוזיקה פעילה) ← ברכת פתיחה (ברירת מחדל אם אין תוכן אחר)

מסכי חירום שתופסים את כל המסך: הודעה דחופה, שבת (זמני הדלקת נרות/הבדלה מחושבים מקומית).
